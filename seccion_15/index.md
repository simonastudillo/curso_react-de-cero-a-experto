# Sección 15: Context API - Búsquedas y favoritos

En esta sección terminaremos la aplicación (búsquedas y favoritos) para hacer los favoritos repasaremos useContext y guardaremos en localStorage y usaremos el contexto para guardar todo lo relacionado con los favoritos.

Todos los filtros y búsquedas se gestionarán a traves de la URL para poder compartir enlaces y mantener el estado al recargar la página.
